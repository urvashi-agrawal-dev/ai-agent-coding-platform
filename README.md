# DevMentor-360

A full-stack multi-agent development mentoring platform with real-time code analysis, debugging, and testing capabilities.

## Features

- **Multi-Agent Architecture**: Architect, Debugger, Reviewer, Tester, and Productivity agents
- **Code Editor**: Monaco-based editor with syntax highlighting
- **File Management**: Upload and manage project folders
- **Sandbox Execution**: Safe code execution environment
- **Testing Integration**: Jest-based testing engine
- **Documentation Generator**: Automatic project documentation
- **Real-time Debug Reports**: Live debugging feedback

## Quick Start

```bash
# Install dependencies
npm run install:all

# Run development servers
npm run dev
```

Frontend: http://localhost:3000
Backend: http://localhost:5000

## Architecture

- **Frontend**: React + TypeScript + Monaco Editor
- **Backend**: Node.js + Express + TypeScript
- **Agents**: Multi-agent orchestration system
- **Sandbox**: Isolated code execution environment
